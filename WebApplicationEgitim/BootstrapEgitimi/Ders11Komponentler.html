Bootstrap Komponentler

Bootstrap web uygulamalarımızda kullanabileceğimiz birçok hazır komponent sunar. Örneğin resimleri sırayla kaydıran bir slider yapmak için carousel komponenti vardır.

Komponentleri kullanabilmek için sayfada css kütüphanesiyle birlikte Bootstrap javascript kütüphanesini de kullanmamız gerekir.

CDN Kullanımı

Web uygulamaları internet üzerinden çalıştığı için dilersek kütüphaneleri uygulama içerisine eklemeden web üzerindeki belirli serverlardan çekerek kullanabilirz.

Bu serverlara kısaca CDN denir ve bize css, js vb kütüphaneleri kolayca kullanma imkani sunar.

Javascript Giriş

Javascript tarayıcı üzerinde çalışabilen bir programlama dilidir.

Javascript kodu yazarken pure veya native denilen ham Javascript kodu ile programlama yapabileceğimiz gibi sonradan Javascript ile geliştirilmiş birçok kütüphaneyi kullanarak daha az kodla aynı Javascript işlemlerini yaptırabiliriz.

Javascript kütüphanelerinden en popülerleri Jquery ve Bootstrap örnek verilebilir.

Javascript ile web programlama, SPA(Sİngle page application), mobil uygulama, oyun, masaüstü programlama gibi pek çok uygulama geliştirilebilir.

Son zamanlarda özellikle SPA(tek sayfalık uygulamalar) popülerdir bu alanda Angular(Google), React(Facebook) ve Vue(topluluk) js kütüphaneleri en sık kullanılanlardır.

Javascript ile Facebook un geliştirdiği React Native kütüphanesini kullanarak Mobil Uygulama geliştirebiliriz.

Javascript kodları herhangi bir sayfa içerisinde script taglarının arasına yazılır.
Sayfada head veya body dilediğimiz alanda Javascript yazabiliriz tarayıcılar script taglarını gördüğü anda kodları çalıştırır.

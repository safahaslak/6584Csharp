Javascript Operatörler

Aritmetik Operatörleri

Matematiksel hesaplamaları yapmak için kullanabileceğimi toplama, çıkarma, çarpma ve bölme gibi operatörler.

Atama Operatörleri

=, +=, -=, *=, /=, %=

Karşılaştırma Operatörleri

== : eşit mi, === : değerler veri tipleriyle birlikte eşit mi, != : eşit değilse, !== değerler ve tipler eşit değilse, > : büyüktür, < : küçüktür, >= : büyük eşittir, <= : küçük eşittir.

Mantıksal Operatörler

&&(and) : ve, ||(or) : veya, ! (not): değil
